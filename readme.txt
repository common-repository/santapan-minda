Show various quotes in malay languange on your dynamic sidebar.

== Description ==

This plugin will grab one quote at a time from SantapanMinda.com, all the quotes are in Malay language. 
Plugin ini akan memaparkan satu Mutiara kata setiap kali halaman dibuka, setiap Mutiara Kata akan diambil terus dari API Santapanminda.com.


== Installation ==

To install this plugin, 

1. Upload `santapan-minda folder` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Appearance > Widgets to add the widget to your sidebar.

Untuk memasang plugin ini,

1. Muat naik folder santapan-minda ke `/wp-content/plugins/` 
1. Aktifkan plugin menerusi menu 'Plugins' dalam WordPress
1. Pergi ke Appearance > Widgets untuk menambah widget ke sidebar anda.
